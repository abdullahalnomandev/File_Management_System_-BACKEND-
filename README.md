# File Package Management System

A comprehensive file package management system built with Next.js, Ant Design, and TypeScript for managing user subscription plans with file type restrictions and folder limits.

## Features

- **Plan Management**: Create, read, update, and delete plan packages
- **File Type Restrictions**: Configure allowed file types (images, videos, audio, documents)
- **Folder Limits**: Set total folders, nesting levels, and files per folder
- **File Size Controls**: Define maximum file sizes for uploads
- **Responsive UI**: Clean and modern interface using Ant Design components
- **Type Safety**: Full TypeScript support for better development experience

## Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org) with App Router
- **UI Library**: [Ant Design 5](https://ant.design)
- **Language**: TypeScript
- **Styling**: CSS Modules / Tailwind CSS
- **HTTP Client**: Custom API fetch utility
- **Icons**: Ant Design Icons
